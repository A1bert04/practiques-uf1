Capítulo 2: Características básicas

Práctica 1 - Primer documento HTML (2.2)

La primera práctica consiste, como su propio nombre indica, en crear el primer documento HTML. Para hacerlo nos ha sido necesario instalar un editor de código (Sublime Text y Visual Studio Code son los mas populares) y posteriormente crear un documento HTML con un nombre al azar.

Acto seguido, hemos copiado el siguiente fragmento de código en el documento y hemos abierto el fichero .html en nuestro navegador.

Código:

<html>
<head>
<title> El primer documento HTML </title>
</head>

<body>
El lenguaje HTML es <strong>tan sencillo</strong> que prácticamente se entiende sin estudiar el significado de sus etiquetas principales.</p>
</body>

</html>

Resultado:

Conocimientos adquiridos con esta practica:

  • Funcionamiento de las etiquetas
  • Etiquetas html, head, body, title, strong y p
  • Distinción entre el head y el body de un documento HTML

Práctica 4 - Elementos en bloque y en linea (2.8)

Hay dos tipos de elementos en HTML: los elementos en bloque y los elementos en línea. Estos son distintos por el espacio que ocupan, ya que los elementos en liena ocupan solo el espacio del texto que contienen, mientras que los elementos en bloque ocupan toda el ancho de la pagina.

En la siguiente imagen podemos observar esta diferencia (añadiendo bordes con CSS):

Código:

<html>
<head>
<title> Ejemplo de elementos en línea y elementos de bloque </title>
</head>

<body>
<p> Los párrafos son elementos de bloque. </p>
<a href="http://www.google.com"> Los enlaces son elementos en línea </a>
<p> Dentro de un párrafo, <a href="http://www.google.com"> los enlaces </a>
siguen siendo elementos en línea. </p>
</body>

</html>

Resultado:

Conocimientos adquiridos:

  • Diferencia entre elementos en bloque y elementos en linea

Capítulo 3: Texto

Práctica 1 - Los parrafos (3.3)

Podemos escribir parrafos en HTML usando la etiqueta p. Los parrafos son elementos en bloque, por lo que ocupan todo el ancho de la pagina.

Código:

<html>
<head>
<title> Ejemplo de texto estructurado con párrafos </title>
</head>

<body>
<p> Este es el texto que forma el primer párrafo de la página. Los párrafos pueden ocupar varias líneas y el navegador se encarga de ajustar su longitud al tamaño de la ventana. </p>

<p> El segundo párrafo de la página también se define encerrando su texto con la etiqueta p. El navegador también se encarga de separar automáticamente cada párrafo. </p>
</body>

</html>

Resultado:

Conocimientos adquiridos con esta practica:

  • Etiqueta p y su uso

Práctica 2 - Las secciones (h) (3.4)

Las secciones son elementos en bloque que sirven para dividir el contenido de una pagina en secciones. Las secciones se definen con las etiquetas h1, h2, h3, h4, h5 y h6. Cada una de estas etiquetas representa una sección distinta, siendo h1 la sección principal y h6 la sección menos importante.

Las etiquetas h solo se usan con el titulo de la sección, y no es necesario ponerla al principio y al final de la sección. Veamos un ejemplo:

Código:

<html>
<head>
<title> Ejemplo de texto estructurado con secciones </title>
</head>

<body>
<h1> Titular de la página </h1>
<p> Párrafo de introducción... </p>
<h2> La primera sub-sección </h2>
<p> Párrafo de contenido... </p>
<h2> Otra subsección </h2>
<p> Más párrafos de contenido... </p>
</body>

</html>

Resultado:

Conocimientos adquiridos con esta practica:

  • Secciones
  • Uso de la etiqueta h

Práctica 6 - Etiquetas ins y del (3.8)

En esta practica entenderemos los usos de las etiquetas ins y del.

La etiqueta del sirve para tachar texto, para que se entenda que ya no tiene validez, en cambio la etiqueta ins subraya el texto, para dar a entender que este es el nuevo texto que lo remplaza.

La siguiente imagen es el resultado del codigo inmediatamente inferior a esta:

Codigo:

<html>
<head>
<title> Ejemplo de etiqueta ins y del </title>
</head>

<body>
<h3> Ejemplo de etiqueta ins y del </h3>
<p> El HTML, acrónimo inglés de Hyper Text Markup Language (lenguaje de <del datetime="20091025" cite="http://librosweb.es/xhtml/capitulo_3/marcado_basico_de_texto.html"> marcado de hipertexto </del> <ins datetime="20091026" cite="http://librosweb.es/xhtml/capitulo_3.html"> marcas hipertextuales </ins> ) es un lenguaje de marcación diseñado para estructurar textos y presentarlos en forma de hipertexto. </p>

</body>
</html>

Capítulo 5: Listas

Práctica 1 - Listas no ordenadas (5.1)

En esta practica entenderemos el uso de las listas no ordenadas.

Las listas no ordenadas son aquellas que no tienen un orden en particular. Para crear una lista no ordenada se utiliza la etiqueta ul (unordered list). Dentro de esta etiqueta se pueden poner las etiquetas li (list item) para cada elemento de la lista.

Copiamos el siguiente codigo en nuestro editor de texto y observamos el resultado:

Código:

<html>
<head><title> Ejemplo de etiqueta UL </title></head>
<body>

<h1> Menú </h1>

<ul>
<li> Inicio</li>
<li> Noticias</li>
<li> Artículos</li>
<li> Contacto </li>
</ul>

</body>
</html>

Resultado:

Conocimientos adquiridos con esta practica:

  • Funcionamiento de las listas desordenadas

Práctica 2 - Listas ordenadas (5.2)

En esta practica entenderemos el uso de las listas ordenadas.

Las listas ordenadas son aquellas que tienen un orden en particular. Para crear una lista ordenada se utiliza la etiqueta ol (ordered list). Dentro de esta etiqueta se pueden poner las etiquetas li (list item) para cada elemento de la lista.

Copiamos el siguiente codigo en nuestro editor de texto y observamos el resultado:

Código:

<html>
<head><title> Ejemplo de etiqueta OL </title></head>
<body>

<h1> Intrucciones </h1>

<ol>
<li>Enchufar correctamente</li>
<li>Comprobar conexiones</li>
<li>Encender el aparato</li>
</ol>

</body>
</html>

Resultado:

Conocimientos adquiridos con esta practica:

  • Funcionamiento de las listas ordenadas

Práctica 3 - Listas de definición (5.3)

En esta practica entenderemos el uso de las etiquetas dt, dl y dd.

Para crear una lista de definición se utiliza la etiqueta dl (definition list). Dentro de esta etiqueta se pueden poner las etiquetas dt (definition term) para cada elemento de la lista. Dentro de esta etiqueta se pueden poner las etiquetas dd (definition description) para cada elemento de la lista.

Copiamos el siguiente codigo en nuestro editor de texto y observamos el resultado:

Código:

<html>
<head><title> Ejemplo de etiqueta DL </title></head>
<body>
<h1> Metalenguajes </h1>

<dl>
<dt> SGML </dt>
<dd> Metalenguaje para la definición de otros lenguajes de marcado_basico_de_texto </dd>

<dt> XML </dt>
<dd> Lenguaje basado en SGML y que se emplea para describir datos. </dd>

<dt> RSS</dt>
<dt> GML</dt>
<dt> XHTML</dt>
<dt> SVG</dt>
<dt> XUL </dt>
<dd> Lenguajes derivados de XML para determinadas aplicaciones </dd>
</dl>

</body>
</html>

Resultado:

Conocimientos adquiridos con esta practica:

  • Funcionamiento de las listas de definiciones

Práctica 1 - Ejercicio (6.1)

En esta practica entenderemos como funciona la inserción de imágenes en una HTML.

Para insertar una imagen en una página HTML, debemos utilizar la etiqueta img y dentro de ella, el atributo src que indica la ruta de la imagen.

Modificar la página de índice del portfolio de los ejercicios del capítulo 4 para mostrar directamente las imágenes de los proyectos.

<link rel="shortcut icon" href="../favicon.ico" type="image/x-icon" />
<title> Mi Sitio </title>
</head>

<body>

<p><a href="portfolio/indice.html" title="Página principal de Mi Sitio" rel="index"> Volver a la pagina principal </a>

Capítulo 7: Tablas

Práctica 1 - Ejemplo de tabla sencilla (7.2)

En esta practica entenderemos como se crean tablas en HTML.

Para crear una tabla en HTML, debemos utilizar la etiqueta table. Dentro de esta etiqueta, debemos incluir la etiqueta tr que indica una fila de la tabla. Por otro lado, dentro de table, debemos usar la etiqueta td para refiernos a cada celda de la tabla.

Replicamos el siguiente codigo y observamos que se crea una tabla con 3 filas y 3 columnas, a parte de la fila de encima.

Código:

<head><title> Ejemplo de tabla sencilla </title></head>
<body>

<h1> Listado de cursos </h1>

<table>
<tr>
<td><strong> Curso </strong></td>
<td><strong> Horas </strong></td>
<td><strong> Horario </strong></td>

<td> HTML </td>

</tr>
