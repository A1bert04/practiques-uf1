Capítulo 2: Características básicas

Práctica 1 - Primer documento HTML (2.2)

La primera práctica consiste, como su propio nombre indica, en crear el primer documento HTML. Para hacerlo nos ha sido necesario instalar un editor de código (Sublime Text y Visual Studio Code son los mas populares) y posteriormente crear un documento HTML con un nombre al azar.

Acto seguido, hemos copiado el siguiente fragmento de código en el documento y hemos abierto el fichero .html en nuestro navegador.

Código:

<html>
<head>
<title> El primer documento HTML </title>
</head>

<body>
El lenguaje HTML es <strong>tan sencillo</strong> que prácticamente se entiende sin estudiar el significado de sus etiquetas principales.</p>
</body>

</html>

Resultado:

Conocimientos adquiridos con esta practica:

  • Funcionamiento de las etiquetas
  • Etiquetas html, head, body, title, strong y p
  • Distinción entre el head y el body de un documento HTML.

Práctica 2 - Ejercicio (2.5)

El ejercicio consiste en imaginar cual es el código HTML de una página por su contenido, a partir de una imagen que nos da el enunciado.

Imagen:

Respuesta:

<html>
<head>
<title> El primer documento HTML </title>
</head>

<body>
<p>
El lenguaje HTML es <strong>tan sencillo</strong> que practicamente se entiende sin estudiar el significado de sus etiquetas principales.
</p>
<p>
Ademas de textos en <strong>negrita</strong>, tambien se pueden poner <em>en cursiva</em> o <del>tachados</del>.
</p>
</body>

</html>

Resultado:

Conocimientos adquiridos:

  • Etiquetas em y del
  • Imaginar el codigo HTML visualizando la pagina
